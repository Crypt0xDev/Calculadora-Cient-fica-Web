# Cómo contribuir

¡Gracias por tu interés en contribuir a este proyecto!

## Reportar un bug

1. Abre un [Issue](https://github.com/Crypt0xDev/Calculadora-Cient-fica-Web/issues/new) con el título claro del problema.
2. Describe los pasos para reproducirlo.
3. Indica el resultado esperado y el resultado obtenido.

## Proponer una mejora

1. Abre un Issue describiendo la funcionalidad que propones.
2. Espera feedback antes de abrir un Pull Request.

## Enviar un Pull Request

1. Haz un fork del repositorio.
2. Crea una rama descriptiva: `git checkout -b feat/mi-mejora`
3. Realiza tus cambios y haz commit: `git commit -m "feat: descripción del cambio"`
4. Abre un Pull Request hacia `main` con una descripción clara.

## Estilo de código

- **Python**: sigue [PEP 8](https://peps.python.org/pep-0008/).
- **JavaScript**: ES6+, sin dependencias externas.
- Los commits deben seguir [Conventional Commits](https://www.conventionalcommits.org/):
  - `feat:` nueva funcionalidad
  - `fix:` corrección de bug
  - `docs:` cambios en documentación
  - `chore:` tareas de mantenimiento
